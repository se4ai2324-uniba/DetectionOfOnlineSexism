Detection Of Online Sexism

Model Description

The Detection Of Online Sexism system is a sophisticated machine learning-based solution designed for the SemEval 2023 - Task 10 - Explainable Detection of Online Sexism (EDOS) challenge. Developed by Grazia Perna, Maria Elena Zaza, and Francesco Brescia, its core function is to identify and analyze sexist content within online textual data. This is achieved through a series of steps, starting with comprehensive data preprocessing, including removal of extra spaces, converting text to lowercase, eliminating punctuation, tokenization, and lemmatization. In addition, this system not only identifies sexist content but also classifies it into nuanced categories, providing a comprehensive tool for addressing sexism in digital spaces. In order to achive all these performance it employs Support Vector Machine (SVM) models and optimizing them through GridSearchCV for hyperparameter tuning.
The system operates in two primary tasks:

  • Task A: Detection Sexism
    • - for detecting whether a post is sexist or not by using the LinearSVC classifier.
  • Task B: Classify type of Sexism
    • - for assigning to each of the sexist texts one of the following categories: threats, derogation, animosity, prejudiced discussions.
